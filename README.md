# etudr
Helping people learn things, constantly.

## Prerequisites
 - node.js
 - npm
 - ionic
 - cordova
 
## Getting started
1. Install all of the above
2. Clone the repository: ```git clone https://github.com/btroo/etudr.git```

To launch the app, cd into the ```app``` directory. Then:
```
ionic serve
```
